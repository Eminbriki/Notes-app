# 🎓 Semester Module Tracker (Notes-app)

A full-stack web application to help students track their university modules, study progress, exam dates, and personal notes.

---

## 🚀 Features

- ➕ Add modules
- 📊 Track progress (percentage)
- 📅 Store exam dates
- 📝 Add notes for each module
- ✏️ Edit modules
- 🗑️ Delete modules
- 🔍 Search modules
- 🌙 Dark mode
- ⚠️ Highlight upcoming exams

---

## 🛠️ Tech Stack

### Backend
- FastAPI (Python)
- SQLite

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)

---

 ## ⚙️ How to Run

### 1. Clone repository

git clone https://github.com/YOUR_USERNAME/Notes-app.git
cd Notes-app .

### 2. Create virtual environment 
python -m venv .venv 

### 3. Install dependencies
python -m pip install fastapi uvicorn

### 4. Run backend
 python -m uvicorn backend.main:app --reload

### 5.Open Frontend
 frontend/index.html Or with live Server
